# SRC Sport Academy 🏆

SRC Sport Academy is a sports management platform designed to help athletes and coaches connect, manage training programs, track performance, and stay updated on academy activities. It provides a simple, efficient, and user-friendly environment that supports sports development and talent growth.

## Features
- Athlete registration and profile management
- Training and activity management
- Performance tracking
- Event and academy updates
- Responsive and user-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
- PHP
- MySQL
